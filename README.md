# Quantum ESPRESSO Tutorial Files  
### Higher Physics by Sanskar Mishra

This repository contains **complete input and output files** used in the **Quantum ESPRESSO (QE)** tutorial series available on the YouTube channel **Higher Physics by Sanskar Mishra**.  
The files are provided in **ZIP format** for easy download and reproducibility of Density Functional Theory (DFT) calculations.
